# Piezoelectric Smart Material Sensor Log Clean-up

A Python toolkit for cleaning and validating voltage data collected from piezoelectric sensors.

## Aim

This project automates the cleaning of raw voltage measurements by detecting and correcting common issues such as missing values, duplicate timestamps, electrical noise, and abnormal spikes.

## Features

- Load sensor logs from CSV files
- Detect missing values
- Remove duplicate timestamps
- Identify voltage spikes
- Visualise raw and cleaned signals
- Export cleaned datasets

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
